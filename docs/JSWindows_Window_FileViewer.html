Class: FileViewer

FileViewer TInst(title, url)

new FileViewer(title, url)

A version of Window used to just view files. Note: the constructor is an asynchronous function (uses the Fetch API), so you need to use await new (JSWindows.Window(...).FileViewer)(...)
Parameters:
Name Type Description
title string The title of the window.
url string The URL of the resource to view.
